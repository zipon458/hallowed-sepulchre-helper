# Hallowed Sepulchre Helper

A RuneLite plugin for the Hallowed Sepulchre agility course in Darkmeyer.

## Features

- **Crossbow Statues** - Highlights active crossbowmen
- **Lightning** - Highlights lightning strike locations
- **Portals** - Highlights active yellow and blue portals
- **Wizard Statues** - Shows fire tiles and incoming fire tiles
- **Bolt & Sword Projectiles** - Highlights tiles bolts and swords

## Customization

All overlays have configurable colors with alpha transparency:
- Fire/incoming fire
- Bolt and sword fill/border
- Lightning fill/border
- Crossbowmen and portal colors
- Border widths and opacity settings
